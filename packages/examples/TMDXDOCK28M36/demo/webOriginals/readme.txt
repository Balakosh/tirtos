ABSTRACT
This file discusses the files used to make the webpages for the demo example.
********************************************************************************

********************************************************************************
default.htm:        Main webpage
layout.css:         Layout file
dspchip.jpg:        Picture of the chip
logobar.gif:        Texas Instruments Logo
jquery.flot.min.js: jQuery file (for the graph)
jquery.min.js:      jQuery file (for the graph)

To convert these to header files, binsrc was ran on each file. binsrc is in the
<tirtos_install_dir>\products\<ndk>\packages\ti\ndk\tools\binsrc directory. 
Refer to the NDK documentation (spru524.pdf) for more details for this tool
and for adding web content in general.
